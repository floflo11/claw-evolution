# Evolution of Claws

An interactive presentation exploring the 530-million-year evolution of claws across the animal kingdom.

## Features

- AI-narrated timeline from Cambrian origins to modern day
- Ambient music powered by Tone.js
- SVG illustrations for each evolutionary era
- Twitter/X-style card presentation format
- Fully responsive design

## Run Locally

```bash
npx serve public
```

Then open [http://localhost:3000](http://localhost:3000).

## Deploy

This project is configured for Vercel. Simply connect the GitHub repo to Vercel and it will deploy automatically.
