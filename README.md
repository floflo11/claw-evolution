# Evolution of Claws

An interactive presentation exploring the 530-million-year evolution of claws across the animal kingdom.

## Features

- AI-narrated timeline from Cambrian origins to modern day
- Ambient music powered by Tone.js
- SVG illustrations for each evolutionary era
- Twitter/X-style card presentation format
- Fully responsive design

## Live Demo

[https://claw-evolution.vercel.app/](https://claw-evolution.vercel.app/)

## Run Locally

```bash
npx serve public
```

## Deploy

This project is configured for Vercel. Simply connect the GitHub repo to Vercel and it will deploy automatically. Add your `OPENAI_API_KEY` environment variable in Vercel project settings for AI narration.
